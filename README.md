# Streamlit RAG

This repository is for the **Streamlit RAG** project. It provides a framework for building Retrieval Augmented Generation (RAG) applications using Streamlit.

## Features
- **Retrieval**: Use a vector database to retrieve relevant information from a collection of documents.
- **Generation**: Use a language model to generate responses based on the retrieved information.
- **Integration**: Integrate with various data sources and models.

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Sree-dutt/streamlit-rag.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
